Подробное описание

Абстрактный класс задает интерфейс для взаимодействия объектов контроллера.

Класс задает интерфейс взаимодействия объектов контроллера при помощи сигналов sendData() и recvData(). Класс содержит чисто виртуальный метод initConnect() для инициализации сигналов и слотов в классах наследниках. В классах наследниках для передачи данных внешним объектам необходимо использовать сигнал sendData(), а для приема данных от внешних объектов необходимо использовать сигнал recvData().
